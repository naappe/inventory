\# 🛍️ Ewity Products Display



A simple HTML page that fetches and displays products from the Ewity API using a CORS proxy.



\## 🔗 Live Demo

\[https://naappe.github.io/inventory/](https://naappe.github.io/inventory/)



\## 🚀 Features

\- Fetches products from Ewity API via CORS proxy

\- Auto-loads on page open

\- Falls back to mock data if API fails

\- Responsive product grid display

\- Product statistics (total, in stock, out of stock)
